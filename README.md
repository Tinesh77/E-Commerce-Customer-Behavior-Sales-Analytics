# E-Commerce-Customer-Behavior-Sales-Analytics
Analyzed 500K+ transaction records using Python (Pandas, NumPy) to uncover sales trends and customer purchasing behavior
Performed data cleaning, transformation, and feature engineering including creation of Revenue, Recency, Frequency, Monetary (RFM) metrics
Designed and implemented RFM Analysis to segment customers into Champions, Loyal, At-Risk, and Lost segments for targeted business strategies
Applied K-Means Clustering (Machine Learning) to identify distinct customer groups based on behavioral patterns
Built optimized MySQL database schema and executed SQL queries for business metrics like revenue trends, top products, and customer insights
Conducted Exploratory Data Analysis (EDA) to identify purchase patterns, order distribution, and customer retention behavior
Developed interactive Tableau dashboards visualizing customer segmentation, cluster analysis, repeat purchase trends, and top customers
Generated actionable insights such as high-value customer contribution, churn risk segments, and peak purchasing behavior to support data-driven decision making
